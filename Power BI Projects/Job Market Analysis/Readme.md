# Job Market Analysis
This is an interactive Power BI dashboard that dashboard provides insights into the data industry, showing the number of professionals per role, average salaries, work-life balance satisfaction, and overall job happiness across countries. It highlights key trends and patterns to support data-driven decisions.

This project answers questions such as:  
What skills are most frequently required?  
How are Data Analyst roles distributed by location and seniority?  
What patterns emerge across job types and work arrangements?
